前提環境

以下が使用できることが前提です。Java 8に関しては2. Java 8の確認で説明します。
使用するJava Editionのバージョンは1.12.2です。Forge導入時に設定します。
  • インターネット環境(ブラウザ等含む)
  • 管理者権限(Windows)またはroot権限(Raspbian) ※環境によっては不要、必要になった場合は適時説明を入れます
  • Java 8
  • Minecraft: Java Edition

導入するもの

以下のModを導入します。
  • Minecraft Forge 1.12.2-14.23.5.2847
  • OptiFine 1.12.2 HD U F5
  • ConnectedTexturesMod MC1.12.2-1.0.30
  • Patchouli 1.1-24
